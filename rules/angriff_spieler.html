Angriff (Spielerfähigkeitstyp)

Einige Spieleffekte und Kartenfähigkeiten beziehen sich auf einen Angriff. Es gibt verschiedene Möglichkeiten, wie ein Angriff stattfinden kann:

  • Ein Held oder Verbündeter kann seine Basiskraft Angriff verwenden, um einen Gegner anzugreifen. Der Charakter muss erschöpft werden, um diese Kraft zu verwenden. Dies fügt dem Gegner Schaden in Höhe des ANG-Wertes des Charakters zu.
    • Ein Charakter kann nur dann einen Basiskraft-Angriff einleiten, falls es einen Gegner gibt, der von diesem Charakter angegriffen werden kann, oder falls der Charakter betäubt ist.
  • Falls eine ausgelöste Fähigkeit als Angriff gekennzeichnet ist – wie z. B. „Held – Aktion (Angriff)“ – gilt das Abhandeln der Fähigkeit als Angriff auf das angegebene Ziel. Sofern im Text der Fähigkeit nicht anders angegeben, wird ein Held beim Verwenden einer solchen Fähigkeit nicht erschöpft.
    • Eine Fähigkeit, die als Angriff gekennzeichnet ist, gilt als einzelner Angriff, auch wenn dieser Angriff mehrere Instanzen von Schaden zufügt.
    • Wird der Schaden einer Angriffsfähigkeit durch eine andere Fähigkeit erhöht, wird jede Instanz von Schaden dieser Angriffsfähigkeit um die entsprechende Menge erhöht. Dies gilt nicht für Instanzen, die das Wort „zusätzlich“ enthalten.
  • Falls eine Fähigkeit den Text hat „Führe der Reihe nach die folgenden X Angriffe durch“ hat, gefolgt von 2 oder mehr Instanzen von Schaden, gilt jede dieser Instanzen als ein eigener Angriff.
    • Eine Fähigkeit, die den Schaden eines Angriffs erhöht, erhört nur den Schaden einen der Angriffe einer Fähigkeit mit mehreren Angriffen und kann für jeden dieser Angriffe separat ausgelöst werden.
  • Angriffe von Helden und Verbündeten können jeden Gegner zum Ziel haben, es sei denn, eine Kartenfähigkeit (wie z. B. Wache) verhindert, dass ein Gegner angegriffen werden kann.
  • Hat ein Angriff mehrere Gegner zum Ziel, wird der angreifende Charakter so behandelt, dass er jeden dieser Gegner angegriffen hat.
    • Jeder angegriffene Gegner mit dem Schlüsselwort Vergeltung X, der den Angriff überlebt hat, fügt seinen Vergeltungsschaden dem angreifenden Charakter zu.
  • Die Reihenfolge der Abhandlung von Fähigkeiten, die durch die Abhandlung eines Angriffs ausgelöst werden, ist wie folgt:
  1. Das Schlüsselwort Vergeltung X (falls der angegriffene Charakter nicht besiegt wurde)
  2. Erzwungene Fähigkeiten mit den folgenden Auslösern (in beliebiger Reihenfolge):
    • „nachdem [Charakter] angegriffen hat [und einen Gegner/Schergen] [beschädigt/besiegt/…] hat
    • „nachdem [Charakter] angegriffen worden ist …“
  3. Freiwillige Fähigkeiten mit den oben aufgeführten Auslösern
  4. Folgeschaden (für Verbündete)
