Fähigkeiten einleiten, Karten spielen

Sobald ein Spieler eine Karte spielen oder eine ausgelöste Fähigkeit einleiten möchte, muss er zunächst seine Absicht erklären. Dann müssen in der angegebenen Reihenfolge zwei Fragen beantwortet werden:

  1. Spieleinschränkungen überprüfen: Kann die Karte zu diesem Zeitpunkt gespielt bzw. die Fähigkeit eingeleitet werden?
    • Falls die Karte 1 oder mehr Ziele angibt, wird überprüft, dass es mindestens 1 gültiges Ziel gibt. Falls es für die Karte oder die Fähigkeit nicht mindestens 1 gültiges Ziel gibt, kann die Karte nicht gespielt oder die Fähigkeit nicht eingeleitet werden.
  2. Kosten bestimmen: Können die Kosten zum Spielen der Karte bzw. zum Einleiten der Fähigkeit unter Berücksichtigung von Modifikatoren bezahlt werden?

Falls beide Fragen mit Ja beantwortet werden können, werden der Reihe nach die folgenden Schritte durchgeführt:

  1. Falls eine Karte gespielt wird, legt der Spieler die Karte offen für alle sichtbar auf den Tisch. (Diese Karte ist nicht im Spiel.)
  2. Alle Modifikatoren auf die Kosten werden angewendet.
  3. Die Kosten werden bezahlt. Falls dieser Schritt erreicht wird und die Kosten nicht bezahlt werden können, wird dieser Prozess ohne Bezahlung irgendwelcher Kosten abgebrochen.
  4. Das Spielen der Karte beginnt bzw. es wird versucht, die Effekte der Fähigkeit einzuleiten.
  5. Die Karte wird gespielt bzw. die Fähigkeit (falls sie im vorherigen Schritt nicht aufgehoben wurde) wird abgehandelt. Die Karte kommt ins Spiel oder, falls es sich um eine Ereigniskarte handelt, ihre Effekte werden abgehandelt und sie wird dann auf den Ablagestapel ihres Besitzers gelegt.
  • Falls einer der obigen Schritte die Auslösebedingung einer Unterbrechungsfähigkeit eintreten lassen würde, darf die Fähigkeit – kurz bevor diese Auslösebedingung eintritt – eingeleitet werden.
  • Falls einer der obigen Schritte die Auslösebedingung einer Reaktionsfähigkeit eintreten lassen würde, darf die Fähigkeit sofort eingeleitet werden, nachdem diese Auslösebedingung eingetreten ist.
  • Falls sich die eingeleitete Fähigkeit auf einer Karte im Spiel befindet und diese Karte während des hier beschriebenen Prozesses das Spiel verlässt, wird der Prozess dadurch nicht gestoppt. Das geschieht nur, falls die Kosten der Fähigkeit nicht mehr bezahlt werden können, weil die Karte das Spiel verlassen hat.
