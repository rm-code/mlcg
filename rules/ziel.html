Marvel Champions - Referenz

Ziel

Falls eine Spielfunktion oder eine Kartenfähigkeit sich auf ein Spielelement fixiert (z.B. auf einen Angriff, der einem Gegner Schaden zufügt), wird das Spielelement während der Abhandlung der Funktion oder der Fähigkeit zum Ziel der Funktion oder der Fähigkeit.

  • Beispiele für mögliche Ziele können unter anderem sein: „der Schurke“, „ein Scherge“, „ein Gegner“, „ein Plan“, „ein Held“, „ein Verbündeter“, „ein Charakter“, „ein Spieler“, „du“, „eine Karte“.

  • Verlangt eine Fähigkeit oder Spielfunktion 1 oder mehr Ziele, kann die Fähigkeit oder Spielfunktion nur eingeleitet werden, falls sie mindestens 1 gültiges Ziel hat. (Beispiel: Eine Fähigkeit mit dem Text „Füge einem Schergen 5 Schaden zu“ kann nicht eingeleitet werden, falls keine Schergen im Spiel sind.)

    • Basiskräfte sind Spielfunktionen, die ein gültiges Ziel verlangen.
    • Die Formulierung „wähle [ein Spielelement]“ gibt an, dass 1 oder mehr Ziele ausgewählt werden müssen, damit eine Fähigkeit eingeleitet werden kann.
    • Fähigkeiten, die dazu führen, dass ein Spieler 1 oder mehr Karten zieht, haben ein gültiges Ziel, solange der Spieler mindestens 1 Karte in seinem Deck hat.

  • Ein Ziel ist für eine Fähigkeit oder Spielfunktion gültig, wenn ein beliebiger Teil der Fähigkeit oder Spielfunktion das Ziel betreffen kann.

    • Beispiele für mögliche Effekte auf Ziele können unter anderem sein: Schaden zufügen/heilen, Bedrohung hinzufügen/entfernen, eine Statuskarte geben/entfernen, ein Ziel erschöpfen / spielbereit machen, ein Ziel besiegen/ ablegen.
    • Ausnahme: Ein Charakter kann auch mit einem ANG-, PLA- oder WID-Wert von 0 eine Aktivierung oder Basiskraft gegen ein Ziel durchführen, das ansonsten ein gültiges Ziel für eine solche Aktivierung oder Basiskraft ist. (Beispiel: Ein Held mit 0 WID kann seine Basiskraft Widerstand gegen einen Plan mit Bedrohung darauf durchführen.)
    • Die Kosten einer Fähigkeit oder Spielfunktion werden nicht berücksichtigt, wenn überprüft wird, ob eine Fähigkeit oder Spielfunktion ein Ziel betreffen kann.
    • Falls eine Fähigkeit oder Spielfunktion mehrere Effekte auf ihr Ziel hat, ist das Ziel gültig, falls mindestens 1 der Effekte das Ziel betreffen kann.
    • Ein Ziel ist für eine Fähigkeit nicht gültig, falls die Fähigkeit dazu führt, dass das Ziel eine Spielfunktion durchführen müsste, die dem Ziel von einer anderen Fähigkeit untersagt wird. (Beispiel: Ein Charakter hat ein Anhang mit dem Text: „Der Charakter mit diesem Anhang kann nicht spielbereit gemacht werden.“ Somit ist dieser Charakter kein gültiges Ziel für eine Kartenfähigkeit, die ihn wieder spielbereit machen kann.)
    • Schaden, der zugefügt, aber nicht genommen wird (wenn Schaden z.B. verhindert wird), gilt, als würde er das Ziel betreffen.
    • Ein Ziel, das „keinen Schaden nehmen kann“, ist kein gültiges Ziel für eine Fähigkeit, die Schaden zufügt.
    • Ein Ziel, das nicht angegriffen werden kann, ist kein gültiges Ziel für eine Fähigkeit, die als Angriff gekennzeichnet ist.
    • Ein Ziel, gegen das kein Widerstand geleistet werden kann, ist kein gültiges Ziel für eine Fähigkeit, die als Widerstand gekennzeichnet ist.
    • Ein Effekt, der „das [Ziel]“ verwendet, bezieht sich auf das Ziel eines vorherigen Effekts und kann nur für ein gültiges Ziel des vorherigen Effekts abgehandelt werden. (Beispiel: Eine Fähigkeit mit dem Text „Füge einem Gegner 5 Schaden zu. Betäube den Gegner.“ kann nur verwendet werden, um einen Gegner zu betäuben, dem Schaden zugefügt werden kann (bzw. der Schaden nehmen kann).

  • Eine Fähigkeit oder Spielfunktion, die mehrere Spielelemente eines bestimmten Typs zum Ziel hat (z.B. „jeder Gegner“), kann eingeleitet werden, wenn sich unter den Spielelementen mindestens 1 gültiges Ziel befindet.

    • Die Fähigkeit oder Spielfunktion wird aber nicht gegen die Spielelemente abgehandelt, die keine gültigen Ziele sind.
    • Beispiel: Das Krisensymbol () verhindert, dass Bedrohung vom Hauptplan entfernt werden kann. Eine Fähigkeit mit dem Text „Entferne 1 Bedrohung von jedem Plan“ kann verwendet werden, auch wenn sich ein Krisensymbol im Spiel befindet, solange es mindestens 1 Plan gibt, von dem Bedrohung entfernt werden. In diesem Fall würde aber keine Bedrohung vom Hauptplan entfernt werden.

  • Eine Fähigkeit, die sich auf ein zukünftiges Ziel bezieht (z.B. auf „die nächste Karte, die du spielst“), benötigt kein Ziel, um eingeleitet zu werden.
