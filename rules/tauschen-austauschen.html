Marvel Champions - Referenz

"Tauschen", "Austauschen"

Die Anweisung, eine Komponente mit einer anderen Komponente zu „tauschen“ (bzw. eine Komponente gegen eine andere Komponente „auszutauschen“), bedeutet, dass die zwei Komponenten miteinander den Ort tauschen.

  • Ein Tausch kann nicht abgeschlossen werden, falls sich nicht an beiden Ort eine Komponente befindet.
    • Beispiel: Du kannst keine „Karte auf deiner Hand mit der obersten Karte deines Decks tauschen“, falls du keine Karten auf der Hand hast.
  • Eine Karte auf deiner Hand mit der obersten Karte des Decks zu tauschen, gilt nicht als das Ziehen der Karte.
  • Wenn du eine Karte in einer Spielzone mit einer Karte in einer nicht im Spiel befindlichen Zone tauschst und diese beiden Karten ...
    • ... den Namen gemeinsam haben, gelten beide Karten nicht, als wären sie ins Spiel gekommen und hätten das Spiel verlassen. Marker, Anhänge und Statuskarten auf der zuvor im Spiel befindlichen Karte werden auf die andere Karte transferiert. Die andere Karte behält auch den Status der zuvor im Spiel befindlichen Karte (spielbereit oder erschöpft).
    • ... keinen Namen gemeinsam haben, gilt die zuvor im Spiel befindliche Karte, als hätte sie das Spiel verlassen, und die zuvor nicht im Spiel befindliche Karte, als wäre sie ins Spiel gekommen. Marker, Anhänge und Statuskarten auf der zuvor im Spiel befindlichen Karte werden nicht auf die andere Karte transferiert. Die andere Karte kommt spielbereit ins Spiel.
