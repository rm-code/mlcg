Verteidigen, Verteidigung

Während eines gegnerischen Angriffs darf ein Spieler gegen diesen verteidigen, indem er Karten verwendet, die er kontrolliert.

  • Es darf immer nur ein Spieler zur gleichen Zeit gegen einen gegnerischen Angriff verteidigen. Solange ein Spieler verteidigt, können andere Spieler nicht gegen denselben Angriff verteidigen.
  • Ein Held kann seine Basiskraft Verteidigung verwenden, um gegen einen gegnerischen Angriff zu verteidigen. Der Held muss erschöpft werden, um diese Kraft zu verwenden. Die durch den Angriff verursachte Menge an Schaden wird um den VER-Wert des Helden gesenkt, und sämtlicher verbleibender Schaden wird dem verteidigenden Helden zugefügt. Solange ein Held gegen einen Angriff verteidigt, können andere befreundete Charaktere nicht gegen denselben Angriff verteidigen.
    • Besagt eine Kartenfähigkeit, für einen Angriff „[einen Helden] zum Verteidiger zu erklären“, wird der Held so behandelt, als würde er eine Basiskraft Verteidigung verwenden.
  • Ein Verbündeter kann erschöpft werden, um gegen einen gegnerischen Angriff zu verteidigen. Schaden durch den Angriff wird dem Verbündeten zugefügt. Solange ein Verbündeter gegen einen Angriff verteidigt, darf kein anderer befreundeter Charakter gegen denselben Angriff verteidigen.
    • Besagt eine Kartenfähigkeit, für einen Angriff „[einen Verbündeten] zum Verteidiger zu erklären“, wird der Verbündete zum Verteidiger des Angriffs.
  • Sobald ein Spieler während eines gegnerischen Angriffs eine ausgelöste Fähigkeit einleitet, die als Verteidigung gekennzeichnet ist – wie z. B. „Held–Unterbrechung (Verteidigung)“ – wird die Identität jenes Spielers zum Verteidiger und so behandelt, als hätte sie gegen den Angriff verteidigt, falls es nicht bereits einen Verteidiger gibt.
    • Die Identität des Spielers gilt ab dem Zeitpunkt als der Verteidiger, wenn mit der Abhandlung der Fähigkeit, die mit Verteidigung gekennzeichnet ist, begonnen wird.
    • Fähigkeiten, die ausgelöst werden, „sobald dein Held gegen einen Angriff verteidigt“, können ausgelöst werden, sobald eine Fähigkeit, die mit Verteidigung gekennzeichnet ist, abgehandelt wird.
    • Das Spielen einer Fähigkeit, die mit Verteidigung gekennzeichnet ist, ist keine Basiskraft Verteidigung und führt nicht dazu, dass die zugefügte Menge an Schaden in Höhe des VER-Wertes gesenkt wird.
    • Wenn der Fähigkeitstext nichts anderes angibt, wird der Held, der eine Fähigkeit verwendet, die mit Verteidigung gekennzeichnet ist, nicht dabei erschöpft.
    • Der verteidigende Spieler darf während eines gegnerischen Angriffs beliebig viele Verteidigung-Fähigkeiten abhandeln (solange ihre Auslösebedingungen erfüllt werden).
    • Hat ein Spieler während eines gegnerischen Angriffs erst einmal eine Fähigkeit abgehandelt, die als Verteidigung gekennzeichnet ist, können andere Spieler während desselben Angriffs keine als Verteidigung gekennzeichneten Fähigkeiten abhandeln.
    • Fähigkeiten, die mit Verteidigung gekennzeichnet sind, können von einem Spieler auch bei Angriffen verwendet werden, bei dem einer seiner Verbündeten verteidigt. In diesem Fall wird die Identität des Spielers nicht zum Verteidiger.
    • Ein Spieler kann Fähigkeiten, die mit Verteidigung gekennzeichnet sind, auch außerhalb von Angriffen auslösen, falls die Auslösebedingungen der Fähigkeit erfüllt sind. Wenn die Fähigkeit auf dieser Weise ausgelöst wird, wird die Identität des Spielers nicht so behandelt, als hätte sie gegen einen Angriff verteidigt.

Falls ein Spieler gegen einen Angriff, der einem anderen Spieler gilt, verteidigt (entweder durch die Basisfähigkeit Verteidigung durch eines von ihm kontrollierten Charakters oder durch das Abhandeln einer Verteidigung-Fähigkeit), wird jener Spieler das neue Ziel des Angriffs. - Jeder ausgelöste Effekt, der eine Form von „du“ enthält, bezieht sich auf den Spieler, der das Ziel des Angriffs ist/war, sobald die entsprechende Fähigkeit abgehandelt wird. (Beispiel: Das „du“ in einer Fähigkeit, die ausgelöst wird, „sobald dich [Gegner] angreift“, bezieht sich auf den Spieler, gegen den der Angriff ursprünglich eingeleitet wurde, während sich das „du“ in der Fähigkeit, die ausgelöst wird, „nachdem dich [Gegner] angegriffen hat“ auf den Spieler bezieht, dessen Charakter gegen den Angriff verteidigt.) - Jede konstante oder Boost-Fähigkeit, die eine Form von „du“ enthält, bezieht sich auf den verteidigenden Spieler.

  • Falls kein Charakter verwendet wird, um gegen einen gegnerischen Angriff zu verteidigen, gilt der Angriff als unverteidigt.Zusätzlich gilt ein Angriff als unverteidigt, wenn der verteidigende Verbündete besiegt wird, bevor Schaden durch den Angriff zugefügt wird (z. B. durch eine „Boost“-Fähigkeit).
  • Fähigkeiten, die ausgelöst werden, nachdem ein Charakter gegen ein Angriff verteidigt hat, werden abgehandelt, nachdem der Angriff beendet worden ist.
    • Falls ein Effekt dazu führt, dass ein verteidigter Angriff endet, bevor er vollständig abgehandelt worden ist, gilt der Angriff noch immer als verteidigt.
    • Falls eine Fähigkeit ausgelöst wird, nachdem ein Charakter eine Basiskraft verwendet hat, wird die Fähigkeit ausgelöst, nachdem ein Angriff abgehandelt worden ist, in dem ein Charakter die Basiskraft Verteidigung verwendet hat.
