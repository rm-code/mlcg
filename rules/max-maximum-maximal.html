Max., Maximum, Maximal

Die Worte „Maximum“ und „maximal“ sowie die Abkürzung „Max.“ legen ein Maximum für alle Exemplare einer Karte (nach Name) für alle Spieler fest.

  • „Max. X pro [Zeitraum]“ legt fest, wie oft Exemplare dieser Karte während des festgelegten Zeitraums maximal gespielt werden dürfen.

    • Falls eine Karte mit einem Maximum aufgehoben wird, wird die Karte dennoch auf das Maximum angerechnet.

  • „Max. X pro Deck“ beschränkt die Anzahl der Exemplare dieser Karte, die in jedem Spielerdeck enthalten sein dürfen.

  • „Max. 1 pro Spieler“ ist spielerspezifisch und beschränkt die Anzahl der Exemplare dieser Karte, die jeder Spieler zu einem bestimmten Zeitpunkt im Spiel kontrollieren darf.

    • Ein Spieler kann nicht die Kontrolle über ein anderes Exemplar einer „Max. 1 pro Spieler“-Karte übernehmen, die er bereits kontrolliert.

  • „Max. 1 pro [Spielelement]“ beschränkt die Anzahl der Exemplare dieser Karte, die an jedes bestimmte Spielelement angehängt werden darf. (Beispiel: An einen Verbündeten kann maximal 1 Exemplar einer Karte mit dem Text „Max. 1 pro Verbündeten“ angehängt werden.)

  • „Max. 1 pro [Instanz]“ beschränkt, wie oft eine Fähigkeit durch eine einzige Instanz eines auslösenenden Effekts auf allen Exemplaren einer Karte mit dieser Beschränkung ausgelöst werden kann. (Beispiel: Nur 1 Karte mit der Fähigkeit mit dem Text „Max. 1 pro Ereignis“ kann pro gespieltem Ereignis ausgelöst werden.)

  • Die Angabe in Klammern „(bis zum einem Maximum von [Wert])“ innerhalb einer Fähigkeit legt fest, wie hoch ein bestimmter Wert für die Dauer der Fähigkeit maximal sein darf.

    • Ein Maximum in der Form von „(bis zu einem Maximum von +X)“ wirkt sich nur auf die Fähigkeit selbst aus. (Beispiel: Mehrere Fähigkeiten mit dem Text „(bis zu einem Maximum von +3 ANG)“ können zusammen verwendet werden, um den ANG des Charakters auf über +3 zu erhöhen.)
    • Ein Maximum in der Form von „(bis zu einem maximalen [Spielelement] von X)“ wirkt sich auf alle Fähigkeiten aus, die ein bestimmtes Spielelement erhöhen. (Beispiel: Bei einer Fähigkeit mit dem Text „(bis zu einem maximalen Handlimit von 7)“ kann der Spieler als Resultat der Fähigkeit nicht mehr als 7 Karten auf der Hand haben.)
